Frequently Asked Questions

Do I need any coding experience? ▼

Not at all! Devory Builder is a 100% visual editor. We handle all the HTML and CSS code generation behind the scenes automatically.

Where can I host the downloaded website? ▼

Because our builder exports pure HTML/CSS/JS, you can host your site absolutely anywhere like Vercel, Netlify, Github Pages, or standard cPanel hosting.

Can I upload my own logo? ▼

Yes! You can instantly upload your brand logo directly from your computer inside the editor, and it will automatically replace the default text logo in the navigation bar.

Do you offer technical support? ▼

Users on our Pro and Agency tiers receive priority 24/7 technical email support. We can help you with anything from domain linking to Formspree form setups.

Do I own the exported code? ▼

Yes! Once you hit Export, the raw HTML and CSS files are completely yours to keep, modify, host, and distribute to your clients forever.

Can I use my own custom domain? ▼

Absolutely. If you upload the exported ZIP file to your web host, you can link it to any custom domain you already own via your domain registrar.
